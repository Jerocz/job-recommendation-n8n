Automatic Job Vacancy Recommender (n8n)
Qué hace
Workflow diario en n8n que:

Toma usuarios activos
Divide en batches
Lee skills del perfil
Calcula top 5 vacantes con score + explicación
Inserta resultados en job_matches
Guarda ranking diario
Cómo importarlo en n8n
n8n → Workflows → Import
Selecciona el archivo: n8n/workflows/automatic-job-vacancy-recommender.json
